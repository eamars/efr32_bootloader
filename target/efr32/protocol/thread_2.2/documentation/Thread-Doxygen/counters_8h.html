Detailed Description

A library to tally up Ember stack counter events.

The Ember stack tracks a number of events defined by ::EmberCountersType and reports them to the app via the emberCounterHandler() callback. This library simply keeps a tally of the number of times each type of counter event occurred. The application must define ::EMBER_APPLICATION_HAS_COUNTER_HANDLER in its CONFIGURATION_HEADER in order to use this library.
