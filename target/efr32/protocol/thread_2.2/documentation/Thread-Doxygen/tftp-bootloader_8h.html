Structs and Packet Types

TftpBootloader is a bootloading protocol. It allows a bootload target to verify an incoming bootload request from a bootload server. The target can hold the server off while it erases its internal storage area for the bootload image.

A tftp-bootloader packet follows the format:

0               1
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
+       + type  +  payload.........
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+

where type is TftpBootloaderPacketType
