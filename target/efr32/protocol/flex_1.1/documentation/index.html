Silicon Labs Flex Documentation

Release Notes / Quick Start Guides

  • Getting Started with the Silicon Labs Flex SDK for the Wireless Gecko (EFR32) Portfolio -- QSG138

    Flex: For the EFR32 portfolio, provides basic information on configuring, building, and installing applications using the Silicon Labs Flex SDK, which offers two paths to application development. One, using Silicon Labs Connect, provides a fully-featured, easily-customizable wireless networking solution. The other begins with Silicon Labs RAIL (Radio Abstraction Interface Layer), and offers a customizable radio interface layer designed to support proprietary or standards-based wireless protocols.

  • Getting Started with the Silicon Labs Flex SDK for the EZR32 -- QSG137

    Flex: For the EZR32, provides basic information on configuring, building, and installing applications with the Silicon Labs Flex SDK, using Silicon Labs Connect, a fully-featured, easily-customizable wireless networking solution.

  • Flex SDK Release Notes

    Flex: Lists compatibility requirements and sources for all software componentsin the development environment. Discusses the latest changes to the SiliconLabs Flex SDK, including added/deleted/deprecated features/API. Reviews bugs that have been fixed since the last release, and any pending ones. Intended for project managers and software developers.

Application Development Fundamentals

  • Fundamentals - Connect -- UG103-12

    Flex: Describes the features and functions of the Silicon Labs Connect stack, including its device types, network topologies, and its 'building block' development methodology using plugins.

  • Fundamentals - RAIL -- UG103-13

    Flex: Describes the features and functions of Silicon Labs RAIL (Radio Abstraction Interface Layer). RAIL provides an intuitive, easily-customizable radio interface layer that is designed to support proprietary or standards-based wireless protocols.

  • Fundamentals - Wireless Networking -- UG103-01

    Flex: Introduces some fundamental concepts of wireless networking. These concepts are referred to in other Fundamentals documents. If you are new to wireless networking, you should read this document first.

  • Fundamentals - Design Choices -- UG103-03

    Flex: Discusses the major decisions to be made about architecting a wireless networking solution, as well as additional decisions to be made if you are designing a ZigBee solution.

More About Connect

  • Silicon Labs Connect User Guide -- UG235

    Flex: Provides in-depth information for developers using the Silicon Labs Connect stack as a foundation for their application development on the Wireless Gecko (EFR32) portfolio or custom hardware. The Connect stack is delivered as part of the Silicon Labs Flex SDK. You should have already installed the Simplicity Studio development environment and the Flex SDK, and be familiar with the basics of configuring, compiling, and flashing Connect-based applications.

  • Silicon Labs Connect API Reference Guide

    Flex: Contains a comprehensive list of APIs used to interface to the Silicon Labs Connect stack.

  • Building Low Power Networks with the Silicon Labs Connect Stack -- AN902

    Flex: Illustrates techniques using the Silicon Labs Connect stack to reduce power consumption in a sensor network developed using the Silicon Labs Connect Stack.

  • Host/Network Co-Processor Communications Using Silicon Labs Thread and Connect -- AN844

    Flex: Describes how to set up and test UART communication between a host and Network Co-Processor (NCP) using ASHv3-UART. You should have a Raspberry Pi, USB cable (for UART communication), and a development board.

  • ASHv3 Protocol Reference -- UG115

    Flex: Describes the protocol used by the UART-based ASHv3 to carry commands and responses reliably between a host processor and a network co-processor.

More About RAIL

  • Silicon Labs RAIL API Reference Guide

    Flex: Contains a comprehensive list of APIs used to interface to the Silicon Labs RAIL library.

  • Flex Gecko 2.4 GHz, 20 dBm Range Test Demo Users Guide -- UG147

    Flex: Provides an easy way to evaluate the link budget of the Wireless Gecko EFR32 devices using Silicon Labs RAIL (RAIL) by performing a range test between two nodes using Range Test, a standalone test application. The range test demo implements Packet Error Rate (PER) measurement.

  • EFR32 Radio Configurator Guide -- AN971

    Flex: Describes the radio configurator GUI for RAIL framework applications. With it, you can create standard or custom radio configurations on which to run your RAIL-based applications. The role of each GUI item is explained.

  • Using RAIL Test

    Flex: Describes the functionality available in the RAILTest application.

  • EFR32 RF Evaluation Guide -- AN972

    Flex: Describes using RAILTest to evaluate radio functionality, as well as peripherals, deep sleep states, etc. With it you can fully evaluate the receiving and transmitting performance and test RF functionality of development kit hardware or custom hardware.
