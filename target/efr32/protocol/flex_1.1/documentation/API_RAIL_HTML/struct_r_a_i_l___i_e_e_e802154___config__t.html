RAIL_IEEE802154_AddrConfig_t* RAIL_IEEE802154_Config_t::addresses

Configure the RAIL Address Filter to allow the given destination addresses.

If addresses is NULL, defer destination address configuration. If a member of addresses is NULL, defer configuration of just that member. This can be overridden via RAIL_IEEE802154_SetAddresses(), or the individual members can be changed via RAIL_IEEE802154_SetPanId(), RAIL_IEEE802154_SetShortAddress(), and RAIL_IEEE802154_SetLongAddress().
