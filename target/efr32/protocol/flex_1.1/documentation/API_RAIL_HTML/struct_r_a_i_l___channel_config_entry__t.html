Detailed Description

Channel configuration entry structure.

Defines a base frequency and channel space and the channel indexes that are valid within this range.

  • frequency = baseFrequency + channelSpacing * (channel - channelNumberStart);

Each RAIL_ChannelConfigEntry_t should not span more than 64 channels.
