uint8_t  rxPaCtuneValue

Set PACTUNE value for TX and RX mode.

This value can vary depending on band and match and board design.

Parameters
[in]txPaCtuneValueTransmit value for pa ctune
[in]rxPaCtuneValueReceive value for pa ctune
Note
PACTUNE will reset to default values when RADIO_PA_Init() or RAIL_RadioConfig() are called.
Warning
The radio should not be transmitting when this function is called!
void PA_PowerLevelOptimize ( int32_t  power)

Optimize the PA settings based on expected output power level.

This API optimizes the current consumption of the radio based on the provided output power. This is only necessary when output power is controlled by PA_PowerLevelSet().

Parameters
[in]powerPower value in dBm * PA_SCALING_FACTOR
Warning
The radio should not be transmitting when this function is called!
uint16_t PA_PowerLevelSet ( uint8_t  pwrLevel,
uint8_t  boostMode 
)

Set the output power level based on power steps available in the chosen PA.

Each PA has distinct maximum power, minimum power, and power step sizes. This API allows direct access to these power steps to tune between the maximum and minimum output power the selected PA is capable of.

Parameters
[in]pwrLevelOutput power level. Note that the maximum power level will change depending on PA selection.
[in]boostModeOutput boost mode. Some PA selections have a mode that will increase the output power for each step if this is enabled.
Returns
MSB Configured boost mode.
LSB Configured power level
Warning
The radio should not be transmitting when this function is called!
