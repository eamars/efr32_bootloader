Introduction

The Ember ZigBee Radio Frequency for Consumer Electronics (RF4CE) Application Framework provides customers with robust implementations of RF4CE standards. Furthermore, it enables users to build custom applications on top of these implementations using application framework entities such as, but not limited to, callbacks (see Ember RF4CE Application Framework Callbacks), a command line interface, sample applications, and architecture-agnostic plugins.
