void emberAfPluginRelativeHumidityMeasurementServerSetMeasurementInterval ( uint32_t  measurementRateS)

Set the hardware read interval.

This function will set the amount of time to wait (in seconds) between polls of the humidity sensor. This function will never set the measurement interval to be greater than the plugin specified maximum measurement interval. If a value of 0 is given, the plugin specified maximum measurement interval will be used for the polling interval.
