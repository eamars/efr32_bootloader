Detailed Description

A library to tally up Ember stack counter events.

The Ember stack tracks a number of events defined by EmberCounterType and reports them to the app via the emberCounterHandler() callback. This library simply keeps a tally of the number of times each type of counter event occurred. The application must define ::EMBER_APPLICATION_HAS_COUNTER_HANDLER in its CONFIGURATION_HEADER in order to use this library.

See counters-ota.h for the ability to retrieve stack counters from a remote node over the air.

Variable Documentation

uint16_t emberCounters[EMBER_COUNTER_TYPE_COUNT]

The ith entry in this array is the count of events of EmberCounterType i.

uint16_t emberCountersThresholds[EMBER_COUNTER_TYPE_COUNT]

The ith entry in this array is the count of threshold values set for the corresponding ith event in emberCounters. The default value is set to 0xFFFF and can be changed by an application by calling emberSet

uint16_t emberMultiNetworkCounters[EMBER_SUPPORTED_NETWORKS][MULTI_NETWORK_COUNTER_TYPE_COUNT]

The value at the position [n,i] in this matrix is the count of events of per-network EmberCounterType i for network n.
