Detailed Description

This group describes the CLI commands for the Window Covering cluster. Listed below is a description of the cluster:

Provides an interface for controlling and adjusting automatic window coverings.

#define EMBER_AF_DOXYGEN_CLI_COMMAND_WINDOW_COVERING_UP
