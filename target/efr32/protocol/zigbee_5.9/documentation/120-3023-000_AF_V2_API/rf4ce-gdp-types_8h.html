typedef void(* EmberAfRf4ceGdpHeartbeatCallback)(uint8_t, EmberAfRf4ceGdpHeartbeatTrigger)

RF4CE GDP heartbeat callback. Any module can subscribe to incoming heartbeat commands by using the emberAfRf4ceGdpSubscribeToHeartbeat() API. The first parameter is the pairing index, the second parameter is the heartbeat trigger.
