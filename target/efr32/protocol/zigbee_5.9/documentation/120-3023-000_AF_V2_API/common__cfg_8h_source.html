145 * (b) When DISABLED, clock is maintained internally via software (see also 'clk.h Note

151 *
152 * (a) When ENABLED, clock is signaled by application calls to Clk_SignalClk().
153 *
154 * (b) When DISABLED, clock is signaled by OS-dependent timing features.
155 *
156 * (3) Configure CLK_CFG_SIGNAL_FREQ_HZ to the number of times the application will signal the
157 * clock every second. CLK_CFG_SIGNAL_FREQ_HZ configuration is required only if
158 * CLK_CFG_SIGNAL_EN is enabled.
159 *********************************************************************************************************
