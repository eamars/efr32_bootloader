Sends a Publish Top Up Log command.

Parameters

srcEndpointThe endpoint of the sender that supports the prepayment cluster.
dstEndpointThe endpoint on the destination node that supports the prepayment client.
commandIndexIndicates which payload fragment is being sent. Set to 0xFE if this is the last expected command.
totalNumberOfCommandsIf the payload does not fit into 1 command, this indicates the total number of sub-commands in the message.
topUpPayloadAn array of Tup Up Payload structures, containing the payload that should be included in the Publish Top Up Log command.
