)

Schedules a prepayment mode at some current or future time.

Parameters
endpointThe endpoint of the device that supports the prepayment server.
providerIdA unique identifier for the commodity supplier to whom this command relates.
issuerEventIdA unique identifier that identifies this prepayment mode event. Newer events should have a larger issuerEventId than older events.
dateTimeUtcThe UTC time that indicates when the payment mode change should be applied. A value of 0x00000000 indicates the command should be executed immediately. A value of 0xFFFFFFFF indicates an existing change payment mode command with the same provider ID and issuer event ID should be cancelled.
paymentControlConfigA bitmap that indicates which actions should be taken when switching the payment mode.
