Sends a Publish Prepay Snapshot command, sent in response to the Get Prepay Snapshot command..

Parameters
nodeIdThe short address of the destination device.
srcEndpointThe source endpoint used in the ZigBee transmission.
dstEndpointThe destination endpoint used in the ZigBee transmission.
snapshotIdA unique identifier allocated by the device that created the snapshot.
snapshotTimeThe UTC time when the snapshot was taken.

snapshotCauseA 32-bit bitmap that indicates the cause of the snapshot.
