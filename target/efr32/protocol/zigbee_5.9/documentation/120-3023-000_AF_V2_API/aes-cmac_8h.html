void emberAfPluginAesMacAuthenticate ( const uint8_t *  key,
const uint8_t *  message,
uint8_t  messageLength,
uint8_t *  out 
)

Computes the Cipher-based Message Authentication Code (CMAC) corresponding to the passed message and the passed key.

Parameters
keyThe key to be used to produce the cipher-based authentication code.
messageA pointer to the message for which a cipher-based authentication code should be computed.
messageLengthThe length in bytes of the message to be authenticated.

A pointer to a 4-byte area memory in which the resulting cipher-based authentication code will be stored.
