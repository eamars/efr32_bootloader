71 *
72 * (2) Configure CPU_CFG_TS_64_EN to enable/disable 64-bits CPU timestamp feature.
73 *
74 * (3) Configure CPU_CFG_TS_TMR_SIZE with the CPU timestamp timer's word size: CPU_WORD_SIZE_08
75 * for 8-bit word size, CPU_WORD_SIZE_16 for 16-bit word size, CPU_WORD_SIZE_32 for 32-bit
76 * word size or CPU_WORD_SIZE_64 for 64-bit word size.
77 *
78 * (4) If the size of the CPU timestamp timer is not a binary multiple of 8-bit octets (e.g.
79 * 20-bits or even 24-bits), then the next lower, binary-multiple octet word size SHOULD be
80 * configured (e.g. to 16-bits). However, the minimum supported word size for CPU timestamp
81 * timers is 8-bits. See also 'cpu.h FUNCTION PROTOTYPES CPU_TS_TmrRd() Note #2a'.
82 *********************************************************************************************************
